# Lemonade App

Aplikasi Android sederhana yang dibuat menggunakan Kotlin dan Jetpack Compose. Aplikasi ini mensimulasikan proses membuat limun (lemonade) melalui beberapa langkah interaktif yang dapat dijalankan dengan menekan gambar pada layar.

## Features

* Memetik lemon dari pohon
* Memeras lemon beberapa kali secara acak
* Mengisi gelas dengan limun
* Memulai proses kembali dari awal
* User Interface menggunakan Jetpack Compose

## Technologies Used

* Kotlin
* Android Studio
* Jetpack Compose
* Material Design 3

## How to Run

1. Clone repository ini.
2. Buka project menggunakan Android Studio.
3. Tunggu proses Gradle Sync selesai.
4. Jalankan aplikasi pada emulator atau perangkat Android.
5. Ikuti instruksi pada layar untuk membuat limun.

## Project Structure

* `MainActivity.kt` : Logika utama aplikasi.
* `res/drawable` : Asset gambar lemon dan limun.
* `ui/theme` : Konfigurasi tema aplikasi.
